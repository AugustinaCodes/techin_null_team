# Null Team

Static multi-page website for the Null Team project. Includes responsive layouts, navigation, and a contact form with basic client-side validation.

## Pages
- Home
- About
- Pricing
- Contact

## Tech
- HTML
- CSS
- JavaScript

## Run locally
Open index.html in your browser, or use a simple local server.

## Project structure
- assets/ – images and SVGs
- styles/ – global and page styles
- scripts/ – site JavaScript
